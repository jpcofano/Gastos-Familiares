## Qué hace F4.2.1

### 1. Seed — backfill de `tarjetaCodigo` en TarjetaPago

Todo movimiento `subtipo=TarjetaPago` tiene `resumenTarjetaId` poblado (verificado: 100%, cero huérfanos). El transformer rellena `tarjetaCodigo` del movimiento copiándolo del resumen vinculado (`Tarjetas_Resumen.TarjetaCodigo`). Reportar: 42 TarjetaPago con código tras el backfill, 0 sin código.

### 2. Seed — código canónico en los esperados de tarjeta

Los 8 items de tarjeta de `GastosEsperados` traen nombres viejos en su campo `Tarjeta` ('Mastercard BBVA', 'Visa BBVA', 'Mastercard Galicia', 'Visa Galicia'). El transformer de `itemsEsperados` los mapea al `codigo` canónico de `config.tarjetas` y puebla `tarjetaCodigo` en el item. Construir la tabla de mapeo verificando contra `config.tarjetas` y los códigos reales de `Tarjetas_Resumen` (BBVA-MASTER-BLK, BBVA-VISA-SIG, GAL-MASTER-BLK, GAL-VISA o los que sean — VERIFICAR, no asumir). Si algún nombre no mapea: error explícito del seed, no null silencioso.

### 3. Seed + tipos — campo opcional `matchTexto` en `itemsEsperados`

- Nuevo campo `matchTexto: string | null` en el tipo `ExpectedItem` y el transformer.
- Fuente: por ahora hardcodeado en el transformer para el único caso que lo necesita: el esperado "Micro Rugby" → `matchTexto: 'micro rugby'`. (Cuando exista ABM de esperados en F5+, será editable.)

### 4. Lógica de match del checklist (en `src/`)

Orden de evaluación por esperado:
1. Si el movimiento tiene `itemEsperadoId` poblado → vínculo directo, manda sobre todo.
2. Si el esperado tiene `tarjetaCodigo` → match = `subtipo=='TarjetaPago' && tarjetaCodigo igual && moneda igual`. (NO usar categoria/subcategoria para estos — el código es la identidad.)
3. Si no → clave de F4.2 (`categoria+subcategoria+moneda`, persona solo para ingresos) **más** `matchTexto` si existe: la descripción del movimiento debe contener el texto (case-insensitive, trim).
- La agregación de múltiples movimientos dentro de un match correcto SIGUE siendo el comportamiento esperado (mostrar suma + cantidad).
