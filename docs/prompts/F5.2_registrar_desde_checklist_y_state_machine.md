## 2. State machine — reemplaza `estadoItem`
```ts
type EstadoItem =
  | 'pagado' | 'parcial' | 'automatico'      // cubiertos
  | 'pendiente' | 'vencido'                  // mes en curso, sin match
  | 'programado'                             // mes futuro, sin match
  | 'no_registrado'                          // mes cerrado, sin match (solo manuales)
  | 'no_aplica';                             // la periodicidad no incluye este mes

function estadoItem(item, matches, mesActualStr, mes): EstadoItem {
  if (!aplicaEnMes(item, mes)) return 'no_aplica';

  if (matches.length > 0) {
    const montoMatched = matches.reduce((s, m) => s + Math.abs(m.monto), 0);
    if (item.montoEsperado != null && montoMatched < item.montoEsperado * 0.99) return 'parcial';
    return 'pagado';
  }
  // sin match:
  if (item.pagoAutomatico) return 'automatico';   // corta ANTES de pendiente/vencido, en cualquier mes
  if (mes > mesActualStr) return 'programado';
  if (mes < mesActualStr) return 'no_registrado';
  // mes en curso:
  if (item.diaVencimiento && item.diaVencimiento < new Date().getDate()) return 'vencido';
  return 'pendiente';
}
```
- `aplicaEnMes(item, mes)`: para `mensual` → `true`. Los no-mensuales quedan **previstos** pero
  devuelven `true` por ahora (no hay mes-ancla todavía — ver anotación 4). Implementarlo como switch
  con `mensual` resuelto y el resto con `// TODO: requiere mes base`.
- `parcial`: tolerancia 1% (`* 0.99`) para no marcar parcial por centavos.

## 3. Checklist — "cubierto"
- `cubierto(estado) = estado === 'pagado' || estado === 'automatico'`.
- El contador "X de Y" cuenta `cubierto` (parcial NO cuenta como cubierto).
- `automatico` se muestra **cubierto, con rótulo propio "automático"** (asumido, sin conciliar) y **no
  suma monto aparte** — decisión tomada: solo figura cubierto.
- Orden sugerido: `vencido(0) < pendiente(1) < parcial(2) < no_registrado(3) < programado(4) < automatico(5) < pagado(6) < no_aplica(7)`.

## Definición de hecho (en `/resumen`)
- Un esperado en autodébito sin movimiento figura **Automático** (cubierto), no pendiente/vencido, en mes en curso y en meses pasados.
