## §0 · Alineación con el contrato (contexto para el implementador)

El contrato fija "proponer, medir y mostrar riesgos — nunca prescribir".
Quitar el bloque "Recomendaciones" del Resumen es coherente con eso: la cara
principal muestra estado (hero, semáforos, total); las palancas viven en la
solapa **Plan**, que F9.91 convertirá en opciones medidas.

## §1 · Sacar el bloque "Recomendaciones" del Resumen

En el Resumen (variante A) va: **hero negro → Riesgos principales
(3 semáforos) → Patrimonio total**. **Eliminar** el bloque intermedio
"Recomendaciones" (las 3 cards numeradas). Las palancas siguen viviendo en la
solapa **Plan** (no se borran del todo, solo salen del Resumen).

**Ajuste de wording (contrato):** si en la solapa Plan el título o labels de
ese contenido dicen "Recomendaciones", renombrar a **"Opciones"** — es el
término del contrato y el que F9.91 va a usar ("opciones medidas"). Ningún
texto visible de Patrimonio debe decir "recomendación/recomendamos".

## §2 · Chips de riesgo — fondo neutro (sacar el rosita/ámbar)

Los chips de "Riesgos principales" usaban fondo teñido
(`rgba(220,38,38,.10)` rojo → rosita, `rgba(245,158,11,.12)` ámbar). Cambiar
**ambos a gris neutro** — el color queda solo en el **punto** (dot) que
indica el estado:

```ts
const SEM = {
  verde:    { dot: 'var(--gf-emerald)', tone: 'success', label: 'En banda',    bg: 'var(--gf-emerald-50)' },
  amarillo: { dot: 'var(--gf-out)',     tone: 'warning', label: 'Atención',    bg: 'var(--gf-gray-100)' },  // antes rgba(245,158,11,.12)
  rojo:     { dot: 'var(--gf-expense)', tone: 'danger',  label: 'Concentrado', bg: 'var(--gf-gray-100)' },  // antes rgba(220,38,38,.10)
};
```

- El punto (dot) sigue rojo/ámbar/verde según banda → la señal no se pierde,
  pero el fondo deja de ser rosita.
- Esto afecta también donde se reuse `SEM[b].bg` (solapa Riesgo, variante C
  si existe). Está bien que queden neutros; el dot y el número siguen
  comunicando el estado.
- Nota de contrato: encaja con "semáforos como resumen visual, no alarma" —
  bajar la carga visual del fondo va en la misma dirección.

## Test

1. Resumen de Patrimonio (A): hero → 3 semáforos de riesgo → Patrimonio
   total. Sin bloque de Recomendaciones.
2. Los chips de riesgo tienen fondo gris (no rosita/ámbar); el punto de color

3. Las palancas siguen disponibles en la solapa Plan, tituladas "Opciones"
   (sin la palabra "recomendaciones" en ningún texto visible).
