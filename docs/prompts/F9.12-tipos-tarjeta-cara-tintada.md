# F9.12 — Tipos de tarjeta + cara tintada por marca

> Reglas: doc-first · una PR por fase · sin tocar Firestore/Functions · móvil primero.

## 1. Tipo de tarjeta
Sumar `tipo` a la tarjeta: **Crédito / Débito / Prepaga**.
- Solo **Crédito** tiene ciclo cierre→vencimiento y genera resumen (el visor lista
  estas). Débito impacta directo contra la cuenta; Prepaga/billetera es híbrido.
- Mostrar el tipo como chip en la cara de la tarjeta (ej. "CRÉDITO").

## 2. Cara tintada por marca
Las caras usaban todas el mismo gradiente ink. Tintar por marca para distinguirlas:
- Gradiente `linear-gradient(135deg, <tint> 0%, var(--gf-ink) 100%)`.
- `tint`: Visa → `#1a1f71` (navy); Mastercard Black → `#23252b` (casi negro premium).
  Texto siempre blanco; el chip de red (Visa Signature, Mastercard Black) arriba a la
  derecha. Solo color de fondo — **no** reproducir logos de marca.

Tarjetas reales (semilla): Galicia Visa, Galicia Mastercard Black, BBVA Visa Signature,
BBVA Mastercard Black. PR visual.
