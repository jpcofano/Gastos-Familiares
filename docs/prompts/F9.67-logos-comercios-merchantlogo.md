# F9.67 — Logos de comercios en movimientos (mapa comercio→dominio + `<MerchantLogo>`)

> Base para mostrar el logo de la empresa en los movimientos. Referencia de diseño en el kit:
> `ui_kits/mobile/MobileShell.jsx` → `MerchantLogo`, y `ui_kits/mobile/data.jsx` → `M_COMERCIO_DOMINIOS` + `comercioDominio()`.
> Primer consumidor: **Por descripción** en Inicio (`src/vistas/Dashboard.tsx`). Resumen/Hoy vienen en prompts aparte
> (reutilizan el mismo componente). Solo frontend. Deploy `--only hosting`.

---

## Idea

Brandfetch (ya lo usa `BankLogo`) necesita un **dominio**. Los movimientos traen razón social
("…Norte S.A. (Edenor)"), no dominio. Resolvemos con un **mapa curado comercio→dominio** (match por substring, sin
acentos), **aparte** del diccionario/destinos para no ensuciarlos. Sin match → monograma con la inicial. Nunca se
dibuja el logo a mano; nunca se adivina dominio de un nombre de persona (transferencias P2P).

const norm = (s: string) => (s ?? '').toLowerCase().normalize('NFD').replace(/[\u0300-\u036f]/g, '').trim();

export function comercioDominio(nombre: string | null | undefined): string | null {
  const n = norm(nombre ?? '');

}

## §2 · `src/design-system/components/core/MerchantLogo.tsx` (nuevo)

Igual patrón que `BankLogo` (misma CDN Brandfetch, `VITE_BRANDFETCH_CLIENT_ID` o como esté hoy), pero resolviendo
`nombre→dominio`:

```tsx
import { useState, useEffect } from 'react';
import { comercioDominio } from '../../../datos/comerciosDominios';

export function MerchantLogo({ nombre, size = 30, radius = 8 }: { nombre: string; size?: number; radius?: number }) {
  const cid = /* mismo client id que BankLogo */;
  const dominio = comercioDominio(nombre);
  const src = cid && dominio ? `https://cdn.brandfetch.io/domain/${dominio}/w/120/h/120?c=${cid}` : null;
  const [fail, setFail] = useState(false);
  useEffect(() => setFail(false), [nombre]);
  if (!src || fail) {
    return (
      <span style={{ width: size, height: size, borderRadius: radius, background: 'var(--gf-gray-100)', color: 'var(--gf-gray-500)',
        display: 'inline-flex', alignItems: 'center', justifyContent: 'center', flexShrink: 0, fontSize: Math.round(size * 0.4), fontWeight: 700 }}>
        {(nombre || '?').trim().charAt(0).toUpperCase()}
      </span>
    );
  }
  return <img src={src} alt={nombre} onError={() => setFail(true)}
    style={{ width: size, height: size, borderRadius, objectFit: 'contain', background: '#fff', border: '1px solid var(--gf-gray-150)', flexShrink: 0 }} />;
}
```
Exportarlo en el barrel de `design-system/components`.
