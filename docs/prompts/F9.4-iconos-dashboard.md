# F9.4 — Íconos del Dashboard

> Reglas: doc-first · una PR por fase · sin tocar Firestore/Functions · paridad
> funcional con la legacy · rumbo visual moderno 2026 (ink/esmeralda) · móvil primero.

En el Dashboard (vista Inicio / Mensual), reemplazá los emojis de los KPI por
íconos de `lucide-react`:

- 🧾 Movimientos → `Receipt`
- 📊 Gasto promedio → `BarChart3`
- 🏷️ Categoría top → `Tag`
- 💸 Mov. más alto → `TrendingUp`

El ícono va **antes** del texto del eyebrow: `size` 13–14px, color gris secundario
(token `--gf-gray-400`), contenedor `display:flex; align-items:center; gap:6px`.
No cambies valores, layout ni copy. PR visual aislada.
