# F9.1 — Tokens del design system → app React

> Pegá este mensaje en Claude Code. Adjuntá / copiá el archivo `tokens.css` (incluido en
> este paquete) al repo de la app **antes** de correr el prompt.

## Antes de correr (opción B — copiar archivos)

El design system NO es accesible directamente desde el repo de la app. Por eso:

1. Copiá `tokens.css` (de este paquete) a `src/styles/tokens.css` en el repo de la app.
   Es el archivo consolidado: reúne colors + typography + spacing + status del DS.
2. (Más adelante, en fases de componentes, se copiarán también los componentes del DS a
   `src/design-system/`.)

## Prompt

```
Tarea: integrá los tokens del design system como variables CSS globales.

Ya copié el archivo de tokens en src/styles/tokens.css (custom properties de color,
tipografía, espaciado, radios, sombras y estados). Importalo globalmente (en el entry
de la app, p. ej. main.tsx o el CSS raíz) para que las variables --gf-* / --color-* /
--space-* / --radius-* / --shadow-* estén disponibles en toda la app.

NO cambies lógica ni componentes de negocio todavía. Dejá los montos preparados para
mostrar ARS-eq y USD-eq (no hardcodear una sola moneda) — pero sin refactors grandes aún.

Verificá que la app compila y levanta. Mostrá diff y esperá aprobación.
```
