# F9.23 — Efectivo = alias cosmético de Mercado Pago

> Reglas: doc-first · una PR por fase · móvil primero. PR de **display**, no migra datos.
> El modelo real de Firestore **no cambia**: Efectivo sigue existiendo como medio.

## Contexto
El modelo real (hoja ResumenMes / `movimientos.banco`) tiene **Efectivo** como medio de
pago propio. Para la familia, "Efectivo ≈ Mercado Pago": el efectivo se mueve casi siempre
vía MP. Decisión cerrada: **mantener Efectivo en el modelo** (paridad, cero migración) y
**plegarlo a Mercado Pago por alias cosmético** en la capa de display/agrupación.

Referencia en el kit (`ui_kits/mobile/data.jsx`):
```js
{ id: 'efec', nombre: 'Efectivo', color: '#16a34a', tipo: 'Efectivo', aliasDe: 'mp', oculto: true }
// + resolver:
const medioCanonico = (nombre) => { /* si el medio tiene aliasDe, devuelve el nombre del target */ }
```

## Tarea
1. **Tabla de medios / config de la familia**: marcar Efectivo con un alias a Mercado
   Pago (campo equivalente a `aliasDe: 'mp'`) y bandera de oculto. No borrar el medio.
2. **Resolver `medioCanonico(nombre)`** (o equivalente): si el medio tiene alias, devuelve
   el nombre del medio destino; si no, el mismo nombre. Usarlo en **toda agrupación o
   etiqueta por medio** (desglose diario por banco del Resumen, Dashboard `bancoDominante`,
   chips de medio, etc.) para que los movimientos con `banco: 'Efectivo'` se cuenten y
   muestren bajo **Mercado Pago**.
3. **Pantalla Medios de pago (Perfil)**: filtrar los medios `oculto` → Efectivo no aparece
   como fila propia (no duplicar). Sigue habiendo 4 filas visibles.
4. **Lectura, no escritura**: los movimientos existentes con `banco: 'Efectivo'` quedan
   igual en Firestore. Solo se aliasan al renderizar/agrupar.

## Aceptación
- Un gasto con `banco: 'Efectivo'` aparece sumado y etiquetado como **Mercado Pago** en
  Resumen y Dashboard.
- La pantalla Medios muestra BBVA · Galicia · Personal Pay · Mercado Pago (sin fila
  Efectivo).
- Cero cambios en Firestore (ni reglas, ni docs, ni Functions). Sin regresión de medios.
