# F9.14 — Compartir informe como PDF

> Reglas: doc-first · una PR por fase · móvil primero. Mecanismo elegido: **PDF**.

El botón "Compartir informe" (Dashboard) hoy es placeholder. Implementar export a **PDF**.

## Comportamiento
- Genera un **PDF del informe del período visible** (mes o año, según el toggle) y lo
  ofrece para descargar / compartir (en mobile, vía la hoja de compartir nativa del SO
  con el archivo PDF; en desktop, descarga).
- Contenido del PDF = resumen del período: Neto, Ingresos/Salidas, Por categoría,
  y (si aplica) el detalle por día. En es-AR, montos con el formato del sistema
  (`$` / `U$S`, sin decimales).

## Implementación sugerida
- Generación client-side (sin tocar Functions): `@react-pdf/renderer` o `html2pdf`/
  `jsPDF` sobre una vista de impresión dedicada del informe.
- Nombre de archivo: `gastos-familiares-2026-06.pdf` (período).
- El botón muestra estado de carga mientras genera.

## Alcance
- Solo Dashboard (Inicio). El chip "Compartir" que sacamos de Cargar (F9.16) NO vuelve;
  compartir es una acción del informe, no un formato de carga.

PR visual + util de export (sin tocar Firestore/Functions).
